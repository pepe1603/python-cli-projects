# Python CLI Projects 🚀

Repositorio que contiene múltiples proyectos CLI desarrollados en Python.

## 📂 Proyectos incluidos

- 📹 fb_downloader → Descargador de videos con yt-dlp
- 📝 notes_cli → Aplicación CLI para gestionar notas
- 🔐 password_generator → Generador y gestor cifrado de contraseñas

---

## 🧠 Estructura

Cada proyecto tiene:

- Su propio entorno virtual (`venv`)
- Su propio `requirements.txt`
- Su propio `README.md`

---

## 🚀 Ejecutar desde el Launcher

Desde la raíz del repositorio:

```bash
python launcher.py


## 🐍 Crear alias para activar entornos manualmente

Añadir al ~/.bashrc o ~/.zshrc:

```bash
alias venv='source venv/bin/activate'

Y luego se acriva:

```bash
source ~/.bashrc
